Plataforma escalável para consulta de dados e validação de documentos com isolamento completo entre clientes.

**Stack:** Next.js · NestJS · PostgreSQL · Redis · Docker  
- Arquitetura multi-tenant com isolamento lógico de dados.
- Implementação de microserviços para processamento de alto volume.
- Provisionamento automatizado e gerenciamento de assinaturas.

Aplicativo mobile para consulta de créditos fiscais integrado com APIs governamentais.

**Stack:** Flutter · Spring Boot · Oracle SQL · Firebase  
- **50k+ usuários ativos** com **99.2% crash-free rate**.
- Integração complexa com OAuth 2.0 (gov.br).
- Otimização de performance: **3-5s → <100ms** via cache estratégico e refatoração de queries.
